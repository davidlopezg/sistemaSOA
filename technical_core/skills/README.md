# Skills del Sistema SOA

Esta carpeta contiene skills especializados que el agente puede invocar.

## Skill Registry

| Skill | Descripción | Archivo |
|-------|-------------|---------|
| pi-subagents | Delegar trabajo a sub-agentes | (builtin) |
| workspace-explorer | Explorar Notion/Workspace | (builtin) |
| setup-oauth | Configurar OAuth | (builtin) |

## Uso

Los skills se invocan automáticamente cuando el contexto lo indica.
Consultar `.agent/SYSTEM_PROMPT.md` para triggers.
